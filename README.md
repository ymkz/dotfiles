# dotfiles

## Installation

### macOS

```sh
curl -sSfL https://raw.githubusercontent.com/ymkz/dotfiles/HEAD/script/bootstrap.macos.sh | bash
```

### WSL

```sh
curl -sSfL https://raw.githubusercontent.com/ymkz/dotfiles/HEAD/script/bootstrap.wsl.sh | bash
```
